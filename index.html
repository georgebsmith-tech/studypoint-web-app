Sequences and Series

Each section is accompanied with a video for better understanding

A sequence is can be formally defined as a function in which the domain is the set of natural numbers, i.e the domain is the infinite set \(\Set{1,2,3,4,5,6,...}\).

Sequences are defined in such a way that they have a particular order or general term, which is actually the function which generates the terms of the sequence.

Mathematically, a sequence is a function \(f(x)\) which maps from the set of natural numbers \(N\) to the set of real numbers \(R\) or complex numbers \(C\)
